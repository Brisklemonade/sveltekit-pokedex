## Installation
1.
```bash
git clone https://github.com/Brisklemonade/sveltekit-pokedex.git
```
2. `cd sveltekit-pokedex`
3. `npm i`
4. `npm run dev`

## API
I used the [PokeAPI](https://pokeapi.co/) to create this project.
